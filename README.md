# MTProxy Manager

A simple tool to install, manage, test, and remove MTProto Proxy on Ubuntu/Debian.

## Features
- Install MTProto Proxy
- Manage multiple proxies
- Start, stop, restart, and check status
- Test proxy connection
- Add sponsor tag
- Uninstall installed components

## Requirements
- Ubuntu 22.04+ or Debian 12+
- Python 3.8+
- Root/sudo access

## Install
```bash
bash <(curl -Ls https://raw.githubusercontent.com/Mahersaber2024/MTProxy-Advanced-/main/install.sh)
```

## Uninstall
Run the uninstall option from the menu or:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Mahersaber2024/MTProxy-Advanced-/main/uninstall.sh)
```
