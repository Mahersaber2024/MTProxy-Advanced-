# MTPulse Proxy Manager

🚀 **MTPulse** is a comprehensive management tool for **MTProto Proxy** (Telegram Proxy) on Ubuntu and Debian servers.

Developed with Python, it provides an interactive menu to install, manage, and configure your MTProto proxy with ease.

---

## ✨ Features

- 📥 **Install MTProto Proxy** – Compile and install the official Telegram MTProxy from source.
- ⚙️ **Service Management** – Start, stop, restart, check status, and view logs.
- 📝 **Add Sponsor Tag** – Add an AD tag from `@MTProxybot` to display sponsored messages.
- 🗑️ **Uninstall** – Completely remove the proxy and its files.
- 🎨 **Beautiful Menu** – Colorful, interactive interface similar to `x3-tf`.
- 🌟 **Sponsored by HeySolo** – https://t.me/HeySoloATM

---

## 📋 Requirements

- **OS**: Ubuntu 20.04+ or Debian 11+
- **Python**: 3.8+
- **Root/Sudo**: Required for installation and management
- **Dependencies**: `git`, `make`, `build-essential`, `libssl-dev`, `zlib1g-dev`, `curl`, `wget`, `xxd`, `figlet`
